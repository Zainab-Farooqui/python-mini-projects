# Python Mini Projects

This repository contains a collection of small Python projects created while learning Python programming.

Projects include simple games, utilities, and GUI applications.

## Projects

- Dice Roll Gambling Game
- Love Calculator (Tkinter GUI)
- Number Guessing Game
- QR Code Generator
- Rock Paper Scissors Game
- Encode / Decode Message Tool
- Outfit Decider
- Snake Water Gun Game
- Flappy Bird Game (Pygame)

## Technologies Used

- Python
- Tkinter
- Pygame
- Random Module
- QRCode Library

These projects helped strengthen my understanding of Python logic, randomization, loops, functions, and GUI programming.
